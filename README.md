# 研究ノート：競馬メモ

## 研究の核心

- 牝系が非常に重要である．特に，隔世遺伝として馬が強化される現象は趣深い．適正として必ず反映されるはず．

---


## 備忘録（Musings）

- バルナバは祖母ファレノプシスはG13勝，その父ブライアンズタイムは晩成，長距離型でサンデーと真逆．ファレノプシスは小柄で，産駒はパワーで劣るものの，シャープでクライミングローズが希少な繁殖牝馬．クライミングローズ産駒は注目．
- 角居が負けたら厩舎のせいであると言っていたように馬の能力は予想できても気持ちよく走らせるためには厩舎による馬の体調を上げて万全な状態で出走させることが非常に重要に思える．厩舎や厩務員で買うというのが合理的か．
- 東急(9005)，東急不動産(3289)：渋谷の都市開発が終了．利益回収時期到来．都心集中の傾向は止まらず，不動産価格上昇と沿線の人口増加で増収が見込める．
- 住友不動産(8830)：大崎ガーデンタワーの景観が不気味．吸い込まれそうになる．大崎-大井町間の再開発，渋沢栄一の田園都市構想の実現なるか．
- ワシントンホテル(4691)：アパホテルの買収か．都心，ビジネス街の好立地に出店し，客室タイプを絞ることでコスト削減．2人向けのつなげることが可能なタイプの客室は不安．ビジネスマンの経費精算システムに対応したポイント還元やプランの充実でどこまで．
